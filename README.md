# Ideale Gebäudereinigung — Hauptseite

Astro + Tailwind site for ideale-gebaeudereinigung.de, hosted on Netlify.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:4321

## Build

```bash
npm run build
```

## Deploy

Auto-deploys via Netlify on push to `main`. Configuration in `netlify.toml`.

## Stack

- Astro 5
- Tailwind CSS 3
- TypeScript (strict)
- @astrojs/sitemap
- Netlify Forms

## Structure

- `src/pages/` — routes
- `src/components/` — reusable components
- `src/layouts/` — page layouts
- `src/data/` — site, services, faq data
- `src/lib/` — tracking helpers
- `public/images/services/` — service images
